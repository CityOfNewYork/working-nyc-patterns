Card

Source: src/components/card/card

Card Components are thumbnails of content and may include the following optional elements:

  1. Program label or badge
  2. Plain language headline
  3. Program name
  4. Plain language description
  5. Link to learn more about the program guide
  6. “Apply Now” call-to-action button

Demonstration

Commercial driver training

Actively Recruiting Accommodates Multiple Languages

Red Hook on the Road

Through this program students prepare to work as commercial drivers. For Unemployed New Yorkers and Adults.

  • Job training,  job certification
  • Transportation
  • 4 weeks,  night class schedule
Learn more about Commercial driver training
